# Keyboard Shortcuts

| Shortcut     | Action                                                                              |
| :----------- | :---------------------------------------------------------------------------------- |
| Ctrl+Z       | Undo                                                                                |
| Ctrl+Shift+Z | Redo                                                                                |
| Del          | If there is at least one ROI on the current image, remove the last one in the list. |
